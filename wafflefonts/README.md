# Recoleta Fonts

Place the following Recoleta font files in this directory:

- Recoleta-Thin.ttf
- Recoleta-Light.ttf
- Recoleta-Regular.ttf
- Recoleta-Medium.ttf
- Recoleta-SemiBold.ttf
- Recoleta-Bold.ttf
- Recoleta-Black.ttf

These fonts are referenced in the CSS and needed for the website to display correctly.
